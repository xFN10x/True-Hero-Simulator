# True Hero Simulator

## An actually good fight simulator for Undyne the Undying

True Hero Simulator is a long over-due fight simluator for Undyne in the genocide route. This simulator aims to be really accurate to the actual fight in Undertale.

THS is made in Godot 4.6.

You can play this on <https://ths.xplate.dev>
